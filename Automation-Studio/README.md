# Automation Studio

## Overview

Automation Studio is used to automate marketing processes and data management activities in Salesforce Marketing Cloud. It enables scheduled workflows for importing, filtering, querying, and sending communications.

## Key Components

* Import Activities
* File Transfer Activities
* SQL Query Activities
* Filter Activities
* Send Email Activities
* Verification Activities

## Skills Covered

* Creating Scheduled Automations
* Importing Data Files
* Running SQL Queries
* Data Segmentation
* Automating Email Campaigns

## Project 1: Customer Data Import Automation

### Objective

Automate customer data import and segmentation process.

### Steps

1. Create Data Extensions.
2. Configure Import File Activity.
3. Create SQL Query Activity.
4. Create Filter Activity.
5. Schedule Automation.

### Deliverables

* Automated Data Import Process
* Customer Segmentation
* Scheduled Workflow

## Project 2: Welcome Email Automation

### Objective

Automatically send welcome emails to new customers.

### Steps

1. Import new customer records.
2. Segment active customers.
3. Configure Send Email Activity.
4. Schedule daily automation.

### Deliverables

* Automated Welcome Email Process
* Daily Campaign Execution

## Tools Used

* Automation Studio
* Data Extensions
* SQL Query Activities
* Email Studio

## Outcome

Implemented automated workflows to improve campaign efficiency and reduce manual effort.
