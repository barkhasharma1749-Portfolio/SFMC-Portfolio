# Contact Builder

## Overview

Contact Builder is used to manage customer data and relationships in Salesforce Marketing Cloud. It helps organize contacts, attributes, and data sources for personalized marketing campaigns.

## Key Components

* Contacts
* Attribute Groups
* Data Sources
* Relationships
* Populations

## Skills Covered

* Creating Contact Models
* Linking Data Extensions
* Managing Attribute Groups
* Contact Data Relationships
* Data Management

## Project 1: Customer Data Model Setup

### Objective

Create and manage customer relationships using Contact Builder.

### Steps

1. Create Data Extensions.
2. Define relationships between customer and order data.
3. Create Attribute Groups.
4. Verify data connections.

### Deliverables

* Customer Data Model
* Linked Attribute Groups
* Contact Relationships

## Tools Used

* Salesforce Marketing Cloud
* Contact Builder
* Data Extensions

## Outcome

Implemented a structured customer data model to support personalized and automated marketing campaigns.
