# Mobile Studio

## Overview

Mobile Studio is a Salesforce Marketing Cloud application used to create, manage, and automate SMS and Mobile Messaging campaigns. It enables businesses to engage customers through personalized text messages and mobile communications.

## Skills Covered

* SMS Marketing
* Mobile Messaging
* Subscriber Management
* Keyword Management
* MobileConnect
* MobilePush
* SMS Automation
* Multi-Channel Marketing

---

## Project 1: Promotional SMS Campaign

### Objective

Send promotional offers to subscribed customers.

### Campaign Flow

1. Customer Opt-In
2. Subscriber Data Extension
3. SMS Creation
4. Audience Selection
5. SMS Delivery
6. Campaign Tracking

### Sample SMS

```text
Hi John,

Get 20% OFF on your next purchase.

Use Code: SAVE20

Offer valid till 30 June.

Shop Now!
```

### Outcome

Increased customer engagement and offer redemption rates.

## Mobile Studio Components

### MobileConnect

Used for SMS messaging and text campaigns.

### MobilePush

Used for mobile app push notifications.

### GroupConnect

Used for messaging through supported channels.

### Keywords

Keywords allow customers to subscribe using predefined words.

Example:

```text
Text JOIN to 12345
```

## Interview Questions & Answers

### What is Mobile Studio?

Mobile Studio is a Salesforce Marketing Cloud application used for SMS, Push Notifications, and mobile messaging campaigns.

---

### What is MobileConnect?

MobileConnect is a Mobile Studio feature used to create, manage, and send SMS campaigns.

---

### What is a Keyword in Mobile Studio?

A keyword is a word customers send to a short code or long code to subscribe or interact with a campaign.

Example:

```text
JOIN
START
SUBSCRIBE
```

---

### What is the difference between Email Studio and Mobile Studio?

| Email Studio       | Mobile Studio          |
| ------------------ | ---------------------- |
| Email Campaigns    | SMS & Mobile Campaigns |
| Uses Email Address | Uses Mobile Number     |
| HTML Email Content | Text-Based Messaging   |
| Email Tracking     | SMS Delivery Tracking  |
